深拷贝与浅拷贝

  • 浅拷贝
    是指重新分配一块内存，创建一个新的对象，里面的元素是原对象中子对象的引用。
  • 深拷贝
    是指重新分配一块内存，创建一个新的对象，并且将原对象中的元素，以递归的方式，通过创建新的子对象拷贝到新对象中。

Python协程

协程是实现并发编程的一种方式。
对操作系统而言，线程是最小的执行单元，进程是最小的资源管理单元。无论是进程还是线程，都是由操作系统所管理的。

协程与多线程的区别：

  • 协程是单线程
  • 协程由用户决定，在哪些地方交出控制权，切换到下一个任务

协程的写法更加简洁清晰，把async/await语法和create_task结合来用，对于中小级别的并发需求已经毫无压力。

GIL 全局解释器

GIL（Global Interpreter Lock）全局解释器锁

  • 作用
    每一个Python线程，在Cpython解释器中执行时，都会先锁住自己的线程，阻止别的线程执行。
  • 原因
    • 设计者为了规避类似于内存管理这样的复杂的竞争风险问题
    • Cpython大量使用C语言库，但大部分C语言库都不是原生线程安全的（线程安全会降低性能和增加复杂度）
