Requests

（一）requests请求方式

  • get
  • post
  • put
  • delete
  • head
  • options

1 get请求
requests.get(url)

2 post请求
requests.post(url, data={请求体的字典}

（二） response方法
获取网页的解码字符串

1 response.text
获取网页的HTML字符串
出现乱码使用 response.encoding = ‘utf-8’

2 response.content.decode()
也可以获得网页正确解码之后的字符串

3 获取其他属性
response = requests.get(‘http://www.baidu.com')
获取请求头 response.request.headers
（如何获取请求头：
刷新网页->点击network->点击Doc->点击Headers->查看Reaquest Headers的User-Agent字段，直接复制
->将刚才复制的User-Agent字段构造成字典形式）

获取响应头 response.headers
获取响应cookies response.cookies
获取请求url response.request.url
获取响应url response.url
获取内容
requests.text
是现成的字符串，但有时会显示异常，需要.content手动编码
requests.content
存的是字节码
经常使用requests.content.decode(‘utf-8’)

【Beautiful Soup】

作用：可以从HTML或XML文件中提取数据的Python库
使用方式：from bs4 import BeautifulSoup
（一）解析库
| 解析器 | 使用方法 | 优势 | 劣势 |
| :—–| :—-: | :—-: |:—-:|
| Python标准库 | BeautifulSoup(markup,’html.parser’) | python内置标准库、执行速度适中、文档容错能力强 |Python 2.7.3 or 3.2.2)前的版本中文容错能力差 |
| lxml HTML解析器 | BeautifulSoup(markup,’lxml’) | 速度快、文档容错能力强 |需要安装C语言库|
| lxml XML解析器 | BeautifulSoup(markup,’xml’) | 速度快、唯一支持XML的解析器 |需要安装C语言库|
| html5lib | BeautifulSoup(markup,’html5lib’) | 最好的容错性、以浏览器的方式解析文档、生成H5格式的文档 |速度慢、不依赖外部扩展|

（三）标签选择器

  • 选择元素
    soup.title
    soup.head
    soup.p
  • 获取名称
    soup.title.name
  • 获取属性
    soup.p.sttrs[‘name’]
    soup.p[‘name’]
  • 获取内容
    soup.p.string
  • 嵌套选择
    soup.head.title.string
  • 子节点和子孙节点
    soup.p.contents
  • 父节点和祖先节点
    soup.a.parent
    list(enumerate(soup.a.parents))
  • 兄弟节点
    list(enumerate(soup.a.next_siblings))
    list(enumerate(soup.a.previos_siblings))

（四）标准选择器

  • find_all(name, attrs, recursive, text, **kwargs)
    可以根据标签名、属性、内容查找文档
    soup.find_all('ul')
    soup.find_all(attrs={'id':'list-1'})
    soup.find_all(attrs={'name':'elements'}
    soup.find_all(id='list-1')
    soup.find_all(class_='element')
    soup.find_all(text='Foo')

  • find(name, attrs, recursive, text, **kwargs)

  • find_parants() find_parant()
    寻找父节点

  • find_next_siblings() find_next_sibling()
    寻找兄弟节点

  • find_previous_siblings() find_previous_sibing()

  • find_all_next() find_next()

  • find_all_previous() find_previous

（五） CSS选择器

通过select()直接传入CSS选择器即可完成选择
soup.select('.panel .panel-heading')
soup.select('ul li')
soup.select('#list-2 .element')

(六) 总结

  • 推荐使用lxml解析库，必要时使用html.parser
  • 标签选择筛选功能弱但是速度快
  • 建议使用find()、find_all() 查询匹配单个结果或者多个结果
  • 如果对CSS选择器熟悉建议使用select()
